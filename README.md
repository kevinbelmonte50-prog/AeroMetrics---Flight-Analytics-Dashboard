# AeroMetrics - Flight Analytics Dashboard

AeroMetrics es un dashboard interactivo de análisis de vuelos desarrollado con HTML, CSS, JavaScript y Plotly.js.  
El proyecto simula un sistema de monitoreo aeronáutico con métricas operativas, filtros dinámicos y visualizaciones orientadas a la toma de decisiones.

## Características

- KPIs principales:
  - vuelos totales
  - puntualidad
  - demora promedio
  - cancelaciones
  - pasajeros totales
- Filtros por:
  - aerolínea
  - ruta
  - período de tiempo (7D / 30D / 90D)
- Visualizaciones interactivas con Plotly:
  - vuelos por día
  - puntualidad por aerolínea
  - rutas más frecuentes
  - estado de vuelos
  - demora promedio por hora
- Tabla de vuelos recientes
- Interfaz visual estilo futurista / aeronáutico
- Datos simulados generados dinámicamente

## Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Plotly.js
- Google Fonts

## Objetivo del proyecto

El objetivo de este proyecto fue diseñar un dashboard visualmente atractivo y funcional para practicar:

- visualización de datos
- manipulación de datos en JavaScript
- diseño de interfaces orientadas a dashboards
- filtros dinámicos
- simulación de datos para análisis operativo

## Cómo ejecutarlo

1. Clonar el repositorio:

git clone https://github.com/kevinbelmonte50-prog/aerometrics-dashboard.git

2. Abrir el archivo index.html en el navegador.

No requiere instalación ni dependencias adicionales.
